# 类实现 imooc-ls 命令

## 使用步骤

1. npm install
2. npm link
3. 进入某个文件夹下运行终端命令: imooc-ls
4. 支持参数 -l -a
5. 也支持组合形式 -la

(注意：-l: 以 list 形式展示(包括文件夹权限、所有者、所有者所在组、大小、时间、文件名)
-a: 显示全部，不过滤隐藏文件)
